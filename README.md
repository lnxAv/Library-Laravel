## Installation
<ul>
    <li>Import the project inside your /Herd folder</li>
    <li>Open terminal at folder location for installation</li>
</ul>

# Step 1: Setup .env
```
cp .env.example .env
```
# Step 2: Install Dependencies 
```
composer install
npm install
npm run dev
```
# Step 3: Generate Application Key & Migrate
```
php artisan key:generate
php artisan migrate
```
# Step 4: Open Project
```
herd open
```
